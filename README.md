## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Vanilla CSS (CSS Modules)
- **Language**: TypeScript

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Visit `http://localhost:3000` to see the template showcase.
